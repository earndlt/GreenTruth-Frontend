
# About the Project

GreenTruth is a compliance, procurement, and analytics platform for EAC and energy/environmental product markets. Features robust procurement workflows, analytics dashboards, registry integration, RBAC/SoD, and billing/subscriptions.

---

## Tech Stack

- Vite
- React + TypeScript
- shadcn/ui + Tailwind CSS
- Supabase
- Stripe, Plaid
